Для создания файла requirements.txt с перечнем модулей установленных в проекте, необходимо использовать команду
(windows) в командной строке: pip freeze > requirements.txt
###################
В проекте используется Python 3.11.5 версии
Проект разрабатывался на windows 10 pro
###################
БД - SQLite3
Создаем базу данных:
Что бы создать таблицы из представления необходимо:
Открыть командную строку в корне проекта ( где расположен основной файл приложения - threads.py)
Импортируем из файла (модуля) переменную с приложением
from threads import db, app

Создаем контекст приложения и создаем по классам-представлениям таблицы
with app.app_context():
    db.create_all()  # добавить все представления

###################
В корне сайта обязательно должны быть два файла для правильной работы краулера
links.txt - в этом файле будут записаны все найденные ссылки на сайте (бд наполняется из этого файла)
state.txt - в этом файле будет хранится ссылка метка на каком месте был прерван процесс, для продолжения обхода страниц.
Памятка - эти файлы автоматически очищаются при нажатии кнопки  - Очистить.
Бд будет очищена так же автоматически при нажатии на кнопку - Очистить.
Папка сайта в директории uploads будет удалена.
