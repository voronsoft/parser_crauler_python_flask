Как это работает ?

Это краулер с примесью парсера.
Задача этой программы получить данные с интересующего вас сайта.
Вы вводите две основные ссылки:
1-Домен
2-Cсылка для парсинга (страница с которой нужно начать процес сбора ссылок).
Далее нажимаем кнопку ЗАПУСТИТЬ и начнется процесс обхода ссылок по сайту.
Будут собраны все уникальные ссылки на сайте без повторений.
Ссылки не принадлежащие другому домену (сайту) будут проигнорированы.
Ссылки на файлы и изображения и им подобные, так же будут проигнорированы.
Процесс сбора ссылок зависит от размера сайта.
Поэтому наберитесь терпения !
Если по какой то причине вам пришлось закрыть страницу в процессе сбора ссылок.
Не беда, при запуске страницы снова (необходимо перейти на страницу ПРОЦЕСС)
Процедура сбора ссылок продолжится с того места где была программа прервана или остановлена.
То есть краулер просто продолжит с того места где остановился.
На странице ПРОЦЕССА есть счетчик найденных страниц.
Если счетчик обнаруженных страниц, не изменялся 60 секунд, то скорее всего процесс можно считать законченным.
Немного пояснения про 60 секунд:
Так как программа собирает ссылки несколькими потоками (4 потока) то вероятней всего,
что, если, счетчик не меняется 60 секунд, ссылки уже повторяются при нахождении.
Так же, можно немного понаблюдать за счетчиком, если вы видите что он перестал меняться
можно переходить к следующему этапу.

Этап создания файлов из найденных ссылок
На странице ЗАПИСЬ, вы увидите счетчик записанных страниц в файле links.txt
при нажатии на кнопку: НАЧАТЬ ЗАПИСЬ--->
Начнется процесс создания файлов из найденных на сайте страниц.
Когда все файлы будут созданы появится кнопка ПРОСМОТР ФАЙЛОВ.
При нажатии на кнопку откроется проводник вашей системы,
Тем самым вы увидите список созданных файлов в виде страниц.
Имена файлов будут такими же как сама ссылка на сайте.
Окончания могут быть разных типов:
.HTML - просто страницы
.CSS - файл с стилями
.JS - файл с кодом js
Файлы начало которых ERROR это страницы с ошибками.
Вот в принципе всё.
Если нажать на кнопку СТЕРЕТЬ то все созданные файлы будут так же удалены.

Описание разделов:

  • ПАРСЕР
    • Эта страница служит для ввода данных для сайта с которым будет работать краулер.
    Поле *Адрес домена (обязательно) необходимо правильно заполнить.
    Допустим у вас есть некий адрес ссылка который вы хотите запустить в краулере.
    Важна основная часть с вашей ссылки, это основной домен.
    В конце ссылки, обязательно должен присутствовать слеш - /
    Пример:
    http(s)://какой_то_адрес.com/

    Поле *Ссылка для парсинга необходимо также правильно заполнить.
    Адрес не должен указывать на страницы с окончанием: .css или .js
    Также ссылка не должна указывать на любой целевой файл.
    То есть, не должен быть адрес на изображение или любой другой файл.
    Важно понимать, что ссылка должна указывать на страницу типа
    .html или просто заканчиваться слешем /
    Пример :
    http(s)://какой_то_адрес.com/
    http(s)://какой_то_адрес.com/страница.html/

  • ПРОЦЕС
    • Эта страница служит для отображения количества найденных ссылок на сайте.
  • ЗАПИСЬ
    • Эта страница служит для создания файлов из найденных ссылок на страницах сайта.
  • ОЧИСТИТЬ
    • Эта опция служит для очистки данных которые вы вводили до этого.
    При нажатии на неё все данные будут стерты.
    После этого можно вводить новые данные для запуска обработки ссылок другого сайта.
